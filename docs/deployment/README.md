# Deployment Patterns

This section documents various deployment patterns for SOGo in production environments.

## Topics

- **Docker / Podman** — Containerized deployment with Compose files
- **Bare Metal** — Direct installation on Debian/Ubuntu/RHEL
- **Kubernetes** — Helm charts and orchestration
- **Reverse Proxy** — nginx, Apache, and HAProxy configurations
- **Database** — PostgreSQL tuning and backup strategies
- **ActiveSync** — Configuring and troubleshooting ActiveSync
- **High Availability** — Load balancing and failover setups

## Contributing

Open a PR with your deployment configuration and a brief description of your environment (scale, OS, database backend, authentication provider).
